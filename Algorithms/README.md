# Algorithms and Data Structures

## Artifact

**CS 300 Vector Sorting Project**

## Overview

_This artifact is a C++ program originally created in CS 300. It loads bid data from a CSV file and allows the user to display and sort the records._

## Enhancement

_I improved the program by adding binary search by bid title, runtime measurement using C++ chrono, and defensive checks to prevent sorting or searching when no records are loaded. I also added a new menu option for searching._

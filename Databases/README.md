# Databases

## Artifact

**CS 340 Grazioso Salvare Dashboard**

## Overview

_This artifact is a Python dashboard connected to a MongoDB database containing animal shelter records. The dashboard allows users to filter and view animals based on rescue criteria._

## Enhancement

_I improved the database architecture by adding environment-variable-based configuration, stronger error handling, reusable rescue query methods, improved separation of concerns, and indexing on frequently queried fields._
